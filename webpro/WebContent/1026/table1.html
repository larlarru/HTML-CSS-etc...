table : 표형태의 데이터를 표현
	table 속성 : border = 1
	tr : row, 한줄
	td : 데이터, 한칸
	th : 데이터의 제목
	caption : 테이블의 제목
	
	여러개의 table의 스타일을 각각 다르게 지정하기 위해서 id를 부여
	table th : table의 후손 th를 선택한다.
	table > tr : table의 자식 tr을 선택한다.
