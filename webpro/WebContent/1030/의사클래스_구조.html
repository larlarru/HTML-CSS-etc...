의사코드 : 구조나 상태에 따라서 선택
	구조 : 
	부모를 기준으로 짝수번째 자식요소를 선택 
		td:nth-child(2n) nth-child(even)
	
	부모를 기준으로 홀수번째 자식요소를 선택
		nth-child(2n+1) nth-child(odd)
	부모를 기준으로 지정한 번째의 자식요소를 선택
		nth-child(3) nth-child(1)
td:nth-child(even) { background : lightblue }
td:nth-child(odd) { background : lightgreen }
#tab2 tr:nth-child(2n) { background : lightgray }
#tab2 tr:nth-child(2n-1) { background : lightblue }
