iframe : html 파일안에 html파일을 포함시킨다.
	<iframe name="ff" src="myhome.html"></iframe>
	
	iframe의 name의 값을 a링크의 target속성으로 지정한다.
	<a href="아메리카노.html" target="ff">아메리카노</a>
	
	

	<h2><span>커피메뉴</span></h2>
	
	span태그는 inline요소로 가로세로 크기가 조절 되지 않는다.
	크기를 조절하기 위해서 display : inline-block; 스타일을 지정한다.
	
	span {
		display : inline-block;
		width : 200px;
		height : 50px;
		margin-top : 30px;
		
	}
